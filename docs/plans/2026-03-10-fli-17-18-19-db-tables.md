## Issues
- FLI-17: search_configs table
- FLI-18: scan_runs table
- FLI-19: price_snapshots table

## Research Context

### Codebase State
- `src/flight_watcher/db.py` — complete with `Base(DeclarativeBase)`, `get_engine()`, `get_session()` context manager
- `src/flight_watcher/models.py` — currently only has `FlightResult` dataclass (non-ORM); ORM models go here alongside it
- `alembic/env.py` — imports `Base` from `db.py`, uses `target_metadata = Base.metadata`, sets DB URL programmatically
- `alembic/versions/` — empty (only `.gitkeep`), no migrations yet
- Dependencies already installed: `sqlalchemy>=2.0`, `alembic>=1.13`, `psycopg[binary]>=3.1`

### Project Conventions
- **Enums**: String-based Python `enum.Enum` (e.g., `ErrorCategory`, `CircuitState`)
- **SQLAlchemy style**: 2.0 API with `DeclarativeBase`
- **Tests**: `tests/test_<module>.py`, class-grouped, `MagicMock` for DB sessions, `_make_*()` factory helpers
- **No pytest fixtures/conftest.py** — helpers only

## Decisions Made

1. **ORM models location**: Add to existing `models.py` alongside the `FlightResult` dataclass. No separate files — project is small.

2. **Column style**: Use SQLAlchemy 2.0 `Mapped[]` + `mapped_column()` annotations for type safety. This is the modern SQLAlchemy 2.0 approach matching the project's `DeclarativeBase` usage.

3. **Enums**: Define `ScanStatus` and `SearchType` as string-based Python enums in `models.py`. Use `sqlalchemy.Enum(ScanStatus, native_enum=False)` to store as VARCHAR (portable, no need for PostgreSQL CREATE TYPE migration complexity).

4. **Price column**: `Numeric(10, 2)` for decimal precision. Currency stored as `String(3)` (ISO 4217).

5. **Timestamps**: Server-side defaults using `func.now()` for `created_at`. `updated_at` uses `onupdate=func.now()`.

6. **Single migration**: All three tables in one migration file since they're part of the same epic and have FK relationships.

7. **Autogenerate import**: Add `import flight_watcher.models` to `alembic/env.py` so Base.metadata picks up the new ORM models.

8. **Table names**: Use snake_case plural as specified in issue titles: `search_configs`, `scan_runs`, `price_snapshots`.

9. **Indexes**: Add indexes on frequently queried columns:
   - `search_configs`: composite index on `(origin, destination)`
   - `scan_runs`: index on `search_config_id`, index on `status`
   - `price_snapshots`: index on `scan_run_id`, composite index on `(origin, destination, flight_date)`

## Implementation Tasks

### Task 1: Define enums and ORM models in `models.py`

Add to `src/flight_watcher/models.py`:

```python
import enum
from datetime import date, datetime
from decimal import Decimal

from sqlalchemy import (
    Boolean,
    Date,
    DateTime,
    Enum,
    ForeignKey,
    Index,
    Integer,
    Numeric,
    String,
    Text,
    func,
)
from sqlalchemy.orm import Mapped, mapped_column, relationship

from flight_watcher.db import Base


class ScanStatus(enum.Enum):
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"


class SearchType(enum.Enum):
    ONEWAY = "oneway"
    ROUNDTRIP = "roundtrip"


class SearchConfig(Base):
    __tablename__ = "search_configs"

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
    origin: Mapped[str] = mapped_column(String(3), nullable=False)          # IATA
    destination: Mapped[str] = mapped_column(String(3), nullable=False)      # IATA
    must_arrive_by: Mapped[date] = mapped_column(Date, nullable=False)
    must_stay_until: Mapped[date] = mapped_column(Date, nullable=False)
    max_trip_days: Mapped[int] = mapped_column(Integer, nullable=False)
    active: Mapped[bool] = mapped_column(Boolean, nullable=False, server_default="true")
    created_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False, server_default=func.now())
    updated_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False, server_default=func.now(), onupdate=func.now())

    scan_runs: Mapped[list["ScanRun"]] = relationship(back_populates="search_config")

    __table_args__ = (
        Index("ix_search_configs_origin_dest", "origin", "destination"),
    )


class ScanRun(Base):
    __tablename__ = "scan_runs"

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
    search_config_id: Mapped[int] = mapped_column(ForeignKey("search_configs.id"), nullable=False)
    started_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False, server_default=func.now())
    completed_at: Mapped[datetime | None] = mapped_column(DateTime(timezone=True), nullable=True)
    status: Mapped[ScanStatus] = mapped_column(Enum(ScanStatus, native_enum=False), nullable=False, server_default=ScanStatus.RUNNING.value)
    last_successful_date: Mapped[date | None] = mapped_column(Date, nullable=True)
    error_message: Mapped[str | None] = mapped_column(Text, nullable=True)

    search_config: Mapped["SearchConfig"] = relationship(back_populates="scan_runs")
    price_snapshots: Mapped[list["PriceSnapshot"]] = relationship(back_populates="scan_run")

    __table_args__ = (
        Index("ix_scan_runs_config_id", "search_config_id"),
        Index("ix_scan_runs_status", "status"),
    )


class PriceSnapshot(Base):
    __tablename__ = "price_snapshots"

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
    scan_run_id: Mapped[int] = mapped_column(ForeignKey("scan_runs.id"), nullable=False)
    origin: Mapped[str] = mapped_column(String(3), nullable=False)
    destination: Mapped[str] = mapped_column(String(3), nullable=False)
    flight_date: Mapped[date] = mapped_column(Date, nullable=False)
    flight_code: Mapped[str] = mapped_column(String(20), nullable=False)
    departure_time: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False)
    arrival_time: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False)
    duration_min: Mapped[int] = mapped_column(Integer, nullable=False)
    stops: Mapped[int] = mapped_column(Integer, nullable=False)
    brand: Mapped[str] = mapped_column(String(30), nullable=False)           # LIGHT/STANDARD/FULL/PREMIUM ECONOMY
    price: Mapped[Decimal] = mapped_column(Numeric(10, 2), nullable=False)
    currency: Mapped[str] = mapped_column(String(3), nullable=False)         # ISO 4217
    search_type: Mapped[SearchType] = mapped_column(Enum(SearchType, native_enum=False), nullable=False)
    fetched_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), nullable=False, server_default=func.now())

    scan_run: Mapped["ScanRun"] = relationship(back_populates="price_snapshots")

    __table_args__ = (
        Index("ix_price_snapshots_run_id", "scan_run_id"),
        Index("ix_price_snapshots_route_date", "origin", "destination", "flight_date"),
    )
```

Keep the existing `FlightResult` dataclass unchanged at the top of the file.
